# Peak Bot Control Panel

Привет! Это проект открытого исходного кода (Open Source) для автоматизации и удаленного управления ботом в Minecraft версии **1.21.1**. Проект объединяет в себе мощь движка **Mineflayer** (работающего через Node.js) и классический графический интерфейс на **Python (Tkinter)**.

Панель позволяет как полностью автоматизировать рутинные задачи (добыча алмазов, защита от мобов), так и управлять действиями бота вручную в режиме реального времени.

---

## Основной функционал

* **Панель движения WASD** — ручное управление направлением движения бота, прыжками и приседаниями (sneak) при помощи зажатия кнопок мыши.
* **Авто-майнинг алмазов (Diamond Miner)** — интеллектуальный поиск блоков `diamond_ore` и `deepslate_diamond_ore` в заданном радиусе, автоматический выбор кирки, построение безопасного маршрута с помощью `mineflayer-pathfinder` и вскапывание.
* **Умная боевая система (Attack & Defense)**:
    * **Авто-защита (в фоне):** Бот сканирует окружение и автоматически отбивается мечом от приближающихся врагов в радиусе 4 блоков.
    * **Атака на всех:** Режим зачистки территории со спринтом до ближайшего валидного моба или игрока.
    * **Игнорирование декораций:** Бот имеет встроенный список исключений (`NON_ATTACKABLE`), поэтому он никогда случайно не сломает ваши рамки, стойки для брони, картины или лодки.
* **Интеграция с игровым чатом** — поле ввода позволяет мгновенно отправлять сообщения в игру (поддерживает отправку по нажатию клавиши `Enter`).
* **Трансляция от первого лица (Prismarine Viewer)** — если модуль активен, вы можете открыть браузер по адресу `http://localhost:8080` и смотреть на мир глазами бота прямо из веб-интерфейса.
* **Многопоточный асинхронный лог** — все действия, ошибки, кики и подключения выводятся в удобное текстовое поле без зависания графической оболочки.

## Требования к среде

Так как проект использует Node.js библиотеки для взаимодействия с протоколом Minecraft, вам понадобятся:

1. **Node.js** (версии 18 или выше)
2. **Python** (версии 3.8 или выше)

## Быстрый запуск и установка

### 1. Установка Node.js зависимостей
Перейдите в папку проекта и установите необходимые npm-пакеты, чтобы Mineflayer мог работать с миром игры, путями и камерой:
```bash
npm install mineflayer mineflayer-pathfinder prismarine-viewer vec3
2. Установка Python зависимостей
Установите мост для интеграции JavaScript-кода в Python:

Bash
pip install javascript
(Примечание: Библиотека javascript автоматически настроит фоновый процесс Node.js для выполнения кода бота).

3. Настройка параметров подключения
Откройте файл peak_final_v2.py в любом текстовом редакторе и укажите данные вашего сервера:

Python
HOST = "localhost"       # IP-адрес сервера (или хостинг)
PORT = 25575             # Порт сервера
USERNAME = "Peak_Bot!"   # Никнейм бота
VERSION = "1.21.1"       # Версия игры
4. Запуск проекта
Запустите скрипт через терминал или вашу любимую IDE:

Bash
python peak_final_v2.py
